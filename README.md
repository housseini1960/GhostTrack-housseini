# GhostTrack-housseini

A simple OSINT tool written in Python to track IP addresses, analyze phone numbers, and lookup usernames across social platforms.

## Features
* **IP Address Tracker:** Fetch country, city, ISP, and geographic coordinates.
* **Phone Number Tracker:** Verify validity, extract carrier information and location.
* **Username Tracker:** Scan popular social media platforms for specific handles.

## Installation & Usage

To run this tool on your device (Termux, Linux, or PC), execute the following commands in your terminal:

```bash
# Clone the repository
git clone https://github.com/housseini1960/GhostTrack-housseini.git

# Navigate into the project folder
cd GhostTrack-housseini

# Install the required dependencies
pip install -r requirements.txt

# Run the tool
python main.py
```
